# Tradução WHMCS 6 Português Brasil
Tradução Livre do WHMCS versão 6.x para Português do Brasil
<br>
Basedado no arquivo Português Brasil original do WHMCS versão 6
#Instalação
Para realizar a instalação desta versão do idioma basta realizar o upload da pasta **_overrides_** para a pasta **LANG** na raiz da instalação do WHMCS.
